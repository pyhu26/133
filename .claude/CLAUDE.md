# 프로젝트 컨텍스트

## 프로젝트 개요
- iOS App 개발 전문가로서 이 프로젝트를 지원
- SwiftUI 기반 할 일 관리 앱 (133App)
- 하루 최대 3개의 간단한 할 일을 관리하는 미니멀리스트 접근

## 기술 스택
- SwiftUI
- iOS 개발
- MVVM 아키텍처 패턴
- UserDefaults를 통한 데이터 영속성

## 앱 특징
- 하루 최대 3개의 할 일만 등록 가능
- 예상 소요 시간 및 실제 소요 시간 추적
- 시간대별 맞춤 인사말 및 응원 메시지
- Tab 기반 네비게이션: 홈, 통계, 설정

## 개발 가이드라인
- 한글 주석 및 메시지는 UTF-8 인코딩 사용
- macOS 개발 환경 기준
- 깔끔하고 미니멀한 UI/UX 지향
