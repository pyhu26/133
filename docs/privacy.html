개인정보 처리방침

최종 수정일: 2025년 1월 26일

요약: 133 앱은 사용자의 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되지 않습니다.

1. 개인정보의 수집 및 이용

133 앱은 사용자의 개인정보를 수집하지 않습니다. 앱 사용을 위해 회원가입이나 로그인이 필요하지 않으며, 어떠한 개인 식별 정보도 요구하지 않습니다.

2. 데이터 저장

2.1 로컬 저장

다음 정보는 사용자의 기기에만 저장됩니다:

  • 할 일 목록 및 완료 상태
  • 예상 시간 및 실제 소요 시간
  • 통계 데이터 (완료한 할 일 수, 집중 시간 등)
  • 사용자 설정 (알림, 다크 모드, 응원 메시지 등)
  • 사용자 프로필 (이름 및 프로필 아이콘)

2.2 서버 전송 없음

위의 모든 데이터는 사용자의 기기 내부에만 저장되며, 개발자 또는 제3자의 서버로 전송되지 않습니다.

3. 데이터 접근 권한

3.1 알림 권한

사용자가 알림 기능을 활성화한 경우, 앱은 iOS 알림 권한을 요청합니다. 이는 타이머 완료 알림 및 일일 리마인더를 위해 사용됩니다. 알림 권한은 언제든지 iOS 설정에서 취소할 수 있습니다.

3.2 사진 라이브러리 권한 (선택 사항)

사용자가 프로필 사진을 설정하고자 할 경우, 사진 라이브러리 접근 권한을 요청합니다. 이는 전적으로 선택 사항이며, 선택한 사진은 기기 내부에만 저장됩니다.

4. 데이터 보안

모든 데이터는 iOS의 기본 보안 메커니즘에 의해 보호됩니다. 사용자의 데이터는 해당 사용자만 접근할 수 있으며, 다른 앱이나 외부에서 접근할 수 없습니다.

5. 데이터 삭제

사용자는 언제든지 다음 방법으로 데이터를 삭제할 수 있습니다:

  • 앱 내 설정 메뉴에서 "모든 데이터 삭제" 기능 사용
  • iOS 설정에서 앱 삭제

앱을 삭제하면 모든 데이터가 영구적으로 삭제되며 복구할 수 없습니다.

6. 데이터 내보내기

사용자는 앱 내 "데이터 내보내기" 기능을 통해 자신의 데이터를 JSON 또는 CSV 형식으로 내보낼 수 있습니다. 내보낸 데이터는 사용자가 원하는 위치에 저장할 수 있습니다.

7. 제3자 서비스

133 앱은 제3자 분석 도구, 광고 네트워크 또는 기타 외부 서비스를 사용하지 않습니다.

8. 어린이 개인정보 보호

133 앱은 어떠한 연령대의 사용자로부터도 개인정보를 수집하지 않으므로, 어린이의 개인정보에 대한 특별한 처리가 필요하지 않습니다.

9. 개인정보 처리방침의 변경

본 개인정보 처리방침은 필요에 따라 업데이트될 수 있습니다. 변경사항이 있을 경우 이 페이지를 통해 공지하며, 최종 수정일을 페이지 상단에 표시합니다.
